# ChatPDF 超简单部署指南

## 🎯 从零到运行只需2步！

### 第1步：克隆项目
```bash
git clone <your-repository-url>
cd ChatPDF
```

### 第2步：运行启动脚本

**Windows:**
```bash
start.bat
```

**Linux/Mac:**
```bash
chmod +x start.sh
./start.sh
```

就这样！脚本会显示菜单让你选择版本。

---

## 📋 版本选择菜单

运行脚本后会看到：

```
请选择要启动的版本:
========================================

1. 基础版 (标准功能)
2. Ultimate版 (高级功能) [推荐]
3. Screenshot版 (AI视觉) [最强]
0. 退出

请输入选项 (0-3):
```

### 选项说明

**1 - 基础版**
- 适合：快速体验
- 功能：PDF对话、AI问答、文档摘要
- 安装时间：~2分钟

**2 - Ultimate版** ⭐ 推荐
- 适合：日常使用
- 功能：基础版 + 向量检索 + 流式响应 + 表格提取
- 安装时间：~5分钟

**3 - Screenshot版** 🚀 最强
- 适合：需要分析图表、表格
- 功能：Ultimate版 + AI视觉分析 + 截图功能
- 安装时间：~5分钟
- 特别提示：需要选择支持视觉的AI模型（设置中标有📸）

---

## ⚡ 自动处理

选择版本后，脚本会**自动**完成：

1. ✅ 检查Python和Node.js环境
2. ✅ 创建Python虚拟环境
3. ✅ 安装对应版本的后端依赖
4. ✅ 安装前端依赖
5. ✅ 配置前端使用正确的组件
6. ✅ 启动后端服务（8000端口）
7. ✅ 启动前端服务（3000端口）
8. ✅ 在浏览器中打开应用

**你只需要选择一个数字，剩下的全自动！**

---

## 🎯 启动后

### 访问地址
- 🎨 前端界面: http://localhost:3000
- 🌐 后端API: http://localhost:8000
- 📚 API文档: http://localhost:8000/docs

### 配置API Key

1. 点击右上角设置图标⚙️
2. 选择API提供商（OpenAI、Anthropic等）
3. 输入模型名称
4. 粘贴API Key
5. 开始使用！

**如果选择了Screenshot版（选项3）:**
- 记得选择支持视觉的模型（标有📸图标）
- 例如：gpt-4o、claude-sonnet-4-5、gemini-2.5-pro

## ❓ 常见问题

### Q: 第一次启动很慢？
A: 正常！首次需要下载安装依赖包（2-5分钟）。之后再启动就很快了。

### Q: 提示找不到Python或Node.js？
A: 需要先安装：
- Python 3.8+: https://www.python.org/downloads/
- Node.js 16+: https://nodejs.org/

### Q: 我想换个版本怎么办？
A: 重新运行 `start.bat` 或 `./start.sh`，选择不同的选项即可。

### Q: Screenshot版的截图功能怎么用？
A: 
1. 上传PDF后，看到工具栏的"📷 整页"和"✂️ 区域"按钮
2. 点击截图
3. 在右侧对话框提问
4. AI会分析截图内容

### Q: 如何切换AI模型？
A: 点击右上角设置⚙️，可以随时更换API提供商和模型。
