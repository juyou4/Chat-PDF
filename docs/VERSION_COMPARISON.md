# ChatPDF 版本对比

## 📊 基础版 vs Pro版

| 功能 | 基础版 | Pro版 | 说明 |
|------|--------|-------|------|
| **AI模型支持** | 2个 | 10+ | Pro版支持最新所有主流模型 |
| **界面布局** | 单栏 | 左右分栏 | Pro版左侧PDF预览+右侧对话 |
| **PDF预览** | ❌ | ✅ | Pro版可直接查看原文 |
| **文本选择** | ❌ | ✅ | Pro版支持选择文本直接提问 |
| **缩放翻页** | ❌ | ✅ | Pro版支持PDF缩放和翻页 |
| **模型管理** | 手动输入 | 下拉选择 | Pro版自动加载可用模型 |
| **代码复杂度** | 简单 | 中等 | Pro版功能更丰富 |

## 🚀 功能详细对比

### 1. AI模型支持

**基础版:**
- OpenAI (GPT-3.5, GPT-4)
- Anthropic (Claude)

**Pro版:**
- ✅ OpenAI (GPT-5.1, GPT-4.1, o4-mini等)
- ✅ Anthropic (Claude 4.5 Sonnet, Opus等)
- ✅ xAI Grok (4.1, 4.1 Fast)
- ✅ 字节豆包 (1.5 Pro 256K)
- ✅ 阿里通义千问 (Qwen3-Max)
- ✅ MiniMax (ABAB 6.5, M2)
- ✅ 智谱GLM (4.6, 355B MoE)
- ✅ DeepSeek (V3.2, R1)
- ✅ Moonshot Kimi (K2)
- ✅ Google Gemini (2.5 Pro, Flash)

### 2. PDF处理

**基础版:**
- 上传PDF → 提取文本 → 存储
- 只能通过对话了解内容

**Pro版:**
- ✅ 上传PDF → 提取文本 → 存储 + Base64编码
- ✅ 左侧直接显示PDF内容
- ✅ 支持翻页浏览
- ✅ 支持缩放（50%-200%）
- ✅ 页码显示和导航

### 3. 文本交互

**基础版:**
- 只能在输入框输入问题

**Pro版:**
- ✅ 输入框输入问题
- ✅ 在PDF中选择文本
- ✅ 选择后弹出操作菜单
- ✅ 一键"AI解释"选中文本
- ✅ 一键复制选中文本

### 4. 用户体验

**基础版:**
```
上传PDF → 生成摘要 → 在聊天框提问
```

**Pro版:**
```
上传PDF → 左侧显示原文 → 右侧查看摘要 → 
边看原文边提问 → 选择关键段落直接提问 →
翻页查看不同部分 → 缩放调整阅读大小
```

## 🔄 升级路径

### 从基础版升级到Pro版

1. **后端升级:**
```bash
# 备份原文件
cp chatpdf_backend.py chatpdf_backend_v1.py

# 替换为Pro版
cp chatpdf_pro_backend.py chatpdf_backend.py

# 无需改动requirements.txt
```

2. **前端升级:**
```bash
# 在React项目中
# 将ChatPDF.jsx替换为ChatPDFPro.jsx
cp ChatPDFPro.jsx src/components/ChatPDF.jsx

# 重启开发服务器
npm start
```

3. **数据迁移:**
- 文档存储结构不同，需要重新上传PDF
- API Key配置保留在浏览器localStorage中
